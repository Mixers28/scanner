# Invariants (Non-Negotiables)

- Operate locally and offline by default; no required cloud/backend dependency.
- Monitor system behavior continuously in the background with minimal performance impact.
- Build anomaly decisions from baseline behavior plus explicit fit-for-purpose whitelisting.
- Require user review/approval before broad whitelisting of unknown processes.
- Trigger verification scans for suspicious activity using local AV/signature/rule-based tooling.
- Explain alerts in plain language first; technical detail is secondary, not primary.
- Do not take destructive remediation actions automatically in MVP.
- Keep memory/spec artifacts in plain Markdown tracked by Git.
